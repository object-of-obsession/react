## Компоненты
Реакт кмпонент можно представить как компонент пользовательского интерфейса. Это может быть кнопка, меню, виджет или даже целое приложение)

### Простые
Представляется в виде функций
```jsx
function Hero() {
  return (
    <div className="container">
        <img src="facebook.github.io/react/img/logo.svg">
        <h1>React</h1>
        <p>Библиотека для создания пользовательских интерфейсов</p>
    </div>
  )
}
```
